## CSS_TRANSFORMACIONES_Y_ANIMACIONES

Con una transformacion de CSS podemos modificar el comportamiento visual de uno o varios elementos sin que afecten al resto ni modifiquen la estructura del layout.

Con las animaciones de CSS podemos crear un timeline con 2 o mas fotogramas clave (keyframes). Se pueden modificar todas las propiedades de CSS que necesitamos.

Definimos la animacion con la regla @key y le das a enter y le damos un nombre
